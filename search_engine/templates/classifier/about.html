About the Project

01 Project Context

This application was developed as part of a comprehensive final assignment for the Information Retrieval and Machine Learning modules. It demonstrates the integration of a search engine (Task 1) and a multi-label document classifier (Task 2).

02 Data Collection Process

The dataset was collected through a two-stage process to ensure variety and quality:

  • Stage 1 (Automated): A BFS-based crawler traversed the Coventry University PurePortal to collect academic publication metadata.
  • Stage 2 (Manual): Targeted snippets from BBC, Reuters, and medical journals were collected to create high-quality multi-label ground truth for training.

03 Technical Consistency Requirement

A core technical requirement was ensuring that the Search Engine (Task 1) and the Classifier (Task 2) "speak the same language." Both systems share a unified TextPreprocessor that handles tokenization, stop-word removal, and normalization (Stemming/Lemmatization) identically, ensuring consistency across the entire retrieval pipeline.
